Sobre o site

Este site gera números aleatórios para os principais sorteios das loterias brasileiras.
Recomendamos a utilização desses sorteios para a geração de combinações para serem jogadas nos seguintes jogos:
MEGA-SENA, LOTO-FÁCIL, QUINA e LOTOMANIA

Os jogos contidos neste site não tem nenhum tipo de relação aos jogos reais. Assim, nenhum resultado obtido através deste poderá ser associado com os jogos da Caixa Econômica Federal.
A utilização do sistema de geração de jogos é apenas servir de auxílio na escolha dos números aleatórios para realização de jogos oficiais.

As imagens das loterias foram obtidas do site da Caixa.
