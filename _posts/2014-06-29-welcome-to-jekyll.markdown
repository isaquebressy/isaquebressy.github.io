Este post fica em `_posts`. Para publicar algo novo, crie `YYYY-MM-DD-slug.md` com `title`, `date` e o corpo em Markdown.

Trecho de código de exemplo:

{% highlight ruby %}
def print_hi(name)
  puts "Hi, #{name}"
end
print_hi('Tom')
#=> prints 'Hi, Tom' to STDOUT.
{% endhighlight %}

Documentação: [Jekyll](https://jekyllrb.com/) e [GitHub Pages](https://docs.github.com/pages).
